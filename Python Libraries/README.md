# Python Libraries

This repository contains beginner-friendly examples and notes for three essential Python libraries used in Data Science, Data Analysis, and Data Visualization.

## 📚 Libraries Included

### 🔢 NumPy
NumPy is a library for numerical computing. It provides fast arrays and mathematical functions for efficient data processing.

**Topics**
- Arrays
- Indexing & Slicing
- Mathematical Operations
- Statistical Functions
- Random Module

---

### 📊 Pandas
Pandas is a library for data manipulation and analysis. It helps organize, clean, filter, and analyze datasets.

**Topics**
- Series
- DataFrame
- Reading & Writing CSV Files
- Data Cleaning
- Filtering & Sorting
- GroupBy & Merge

---

### 📈 Matplotlib
Matplotlib is a data visualization library used to create charts and graphs.

**Topics**
- Line Plot
- Bar Chart
- Pie Chart
- Histogram
- Scatter Plot
- Chart Customization

---

## 💻 Installation

Install all required libraries:

```bash
pip install numpy pandas matplotlib
```

Or using Anaconda:

```bash
conda install numpy pandas matplotlib
```
