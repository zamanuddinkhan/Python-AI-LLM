# Python-AI-LLM 🚀

## 📌 Overview

This repository contains structured learning materials and notes from a comprehensive course covering Python fundamentals, Large Language Models (LLMs), Prompt Engineering, Retrieval-Augmented Generation (RAG), and Agentic AI systems. The course focuses on building a strong foundation and applying concepts through practical workflows and real-world use cases.

---

## 🧠 Topics Covered

### 1. 🐍 Python Fundamentals

* Python setup (Anaconda, VS Code, Jupyter)
* Syntax, variables, data types, type casting
* Control flow (if-else, loops, operators)
* Data structures (lists, tuples, dictionaries, sets)
* Functions, lambda expressions, modules & packages
* Object-Oriented Programming (OOP)
* Type hinting and data validation using Pydantic

---

### 2. 🤖 LLM Landscape & Architecture

* Transformer architecture overview
* Self-attention mechanism
* Tokens, embeddings, and tokenization
* Overview of modern LLMs and their capabilities
* Open-source vs hosted models
* Model selection and inference basics
* Cost, latency, and scaling considerations
* Context window limitations and hallucination mitigation
* Fine-tuning and instruction tuning

---

### 3. 📝 Prompt Engineering

* Prompt design principles
* Zero-shot and few-shot prompting
* Role-based prompting techniques
* Advanced methods:

  * Chain-of-thought reasoning
  * Self-consistency
  * Prompt chaining
  * Programmatic prompting pipelines
* Structured outputs and JSON schema
* Function calling concepts
* Prompt injection and adversarial techniques

---

### 4. 📚 Knowledge Augmentation (RAG)

* RAG fundamentals
* Embeddings for retrieval
* Vector databases (FAISS, Chroma)
* Indexing pipelines
* Chunking strategies (fixed, semantic, recursive)
* Metadata handling
* Evaluation using RAGAS framework
* Improving retrieval quality

---

### 5. 🤖 Agentic AI Systems

* Introduction to AI Agents and Agentic AI
* Core concepts: autonomy, reasoning, tool usage
* ReAct paradigm
* Tool-using agents and memory systems
* Short-term and long-term memory
* LangChain fundamentals (chains, tools, memory)
* LangGraph for stateful workflows
* Multi-step reasoning and DAG-based design
* Model Context Protocol (MCP)
* Multi-agent collaboration with CrewAI
* Task orchestration and goal decomposition
* Human-in-the-loop systems
* Deployment best practices
